# Airport Ops Visual v13

V13 resets the Pixi visual direction around a cleaner asset pipeline.

## What changed
- New clean background map: `public/assets/airport/terra01/maps/terminal-base-clean.png`
- New simple runway/apron background: `public/assets/airport/terra01/maps/runway-base-clean.png`
- Backgrounds contain no passengers: NPCs are generated by the renderer.
- Pixi renderer now loads the clean terminal base, not the runway-focus image.
- Added in-map mobile-style HUD overlay for cash, reputation, pax/hour, wait, punctuality and mission.
- Kept mission win/fail loop, stars and localStorage campaign unlock from v12.
- Added `.npmrc` and public npm lock URLs so install works outside the OpenAI environment.

## Run
```bash
npm install --no-audit --no-fund
npm run dev
```

If npm gets stuck because of an old lock, run:
```bash
rm -rf node_modules package-lock.json
npm config set registry https://registry.npmjs.org/
npm install --no-audit --no-fund --registry=https://registry.npmjs.org/
npm run dev
```
