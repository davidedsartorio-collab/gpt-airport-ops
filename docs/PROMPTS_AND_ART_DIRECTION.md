# Airport Ops — Prompt & Art Direction Notes

Usare questo documento quando si generano nuovi asset.

## 1. Prompt master per mappa base

Obiettivo: terminal base senza persone.

```text
Create a clean top-down / three-quarter cartoon airport terminal game map in a vibrant mobile management game style. Use a purple-blue airport floor, dark navy outlines, yellow and blue signage, readable modular zones, and a consistent playful cel-shaded look. The scene must include departures, check-in, bag drop, security, waiting area, boarding gates, baggage claim, cafe/shop, entrance/arrivals, and a small apron edge. No passengers, no staff, no UI overlay, no dynamic vehicles, no weather effects. Keep the camera fixed, 16:9, with clear empty walking paths for game sprites.
```

## 2. Prompt master per NPC sheet

```text
Create a coherent sprite sheet of small rounded airport passenger characters in a crew-like original cartoon style with blue visors, dark outlines, and consistent proportions. Include standard passenger, business traveler, budget traveler, family traveler, VIP traveler, security officer, check-in staff, ground crew. For each, include idle, walk 1, walk 2, waiting, and with luggage states. Use the same camera angle and lighting as the airport terminal map. Transparent or clean simple background, no scene.
```

## 3. Prompt master per aerei e veicoli

```text
Create a coherent game asset sheet of cartoon airport planes and ground vehicles matching the same purple-blue airport game style. Include four small passenger planes with different liveries, baggage train, fuel truck, pushback tug, airport bus, service van, catering truck. Use consistent top-down / three-quarter view, dark outlines, soft shadows, and clean separated assets. No full airport scene.
```

## 4. Prompt master per upgrade modules

```text
Create a clean modular airport game asset sheet in the same cartoon style. Show check-in level 1, 2, 3; security level 1, 2, 3; gates level 1, 2, 3; baggage level 1, 2, 3; cafe/shop level 1, 2, 3. Each module should be separable, with no passengers embedded unless explicitly marked as staff. Same camera angle, same palette, clean layout.
```

## 5. Prompt master per overlay eventi

```text
Create game overlay assets for the same cartoon airport style: rain streaks, lightning flash, fog layer, wet floor reflection, bottleneck glow, delay alert badge, peak hour marker, warning icons. These must be overlays or UI elements, not new full airport scenes. Keep transparent/clean background and consistent palette.
```

## 6. Prompt negativo

Evitare:
- realistic airport blueprint;
- photorealistic;
- random people in background;
- different layout;
- new camera angle;
- full UI baked into image;
- excessive text;
- too much detail that cannot be controlled;
- inconsistent character proportions;
- IP-copying exact characters.

## 7. Regola prompt

Ogni prompt deve sempre dire:

```text
same visual style, same camera angle, same proportions, same palette, no full scene reinterpretation
```
